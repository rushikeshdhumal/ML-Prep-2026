## Folder Expectations

- `dsa/`: topic-wise coding problems + complexity notes
- `sql/`: query sets (easy/medium/hard) + explanation
- `kaggle/`: competition folders with notebook versions and submission logs
- `ml-theory/<course_name>/`: course notes, quizzes, concise summaries
- `mlops/`: ML pipeline, tracking, Docker, CI/CD, monitoring notes
- `system-design/`: ML architecture case studies and trade-offs
- `papers/`: 1-page summaries, critiques, and implementation ideas
- `interview/<topic_name>/`: likely interview Q&A and model answers
- `projects/`: polished end-to-end flagship projects
- `notes/`: daily learning log + mistakes + next-day plan
